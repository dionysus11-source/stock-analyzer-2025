# OpenSpec: CSV 파일 병합 기능 추가

## 1. 제안 배경 (Background)

- 현재 분석에 여러 개의 CSV 파일을 사용하고 있어 데이터 취급이 번거롭습니다.
- 여러 파일에 분산된 데이터를 하나로 합치고, 중복된 내용을 제거하여 데이터 분석의 효율성을 높일 필요가 있습니다.

## 2. 제안 내용 (Proposal)

- 여러 CSV 파일을 사용자가 선택하여 하나의 파일로 병합하는 기능을 추가합니다.
- 병합 과정에서 중복된 행은 제거하며, 원본 파일은 보존됩니다.

## 3. 기능 명세 (Specification)

### 3.1. 사용자 인터페이스 (UI)

- 메인 UI에 `[CSV 파일 병합]` 버튼을 추가합니다.
- 버튼 클릭 시, 시스템 파일 탐색기가 열리며 사용자는 다중 선택(Ctrl/Shift + 클릭)으로 여러 CSV 파일을 선택할 수 있습니다.

### 3.2. 병합 프로세스 (Merge Process)

1.  사용자가 선택한 파일들을 입력으로 받습니다.
2.  `merged_data.csv` 라는 이름의 새로운 결과 파일을 생성합니다. (기존 파일이 있다면 덮어쓰기)
3.  **헤더 처리**: 첫 번째 선택된 파일의 첫 줄(헤더)을 `merged_data.csv`에 씁니다.
4.  **데이터 취합**:
    - 첫 번째 파일의 나머지 데이터 줄을 `merged_data.csv`에 추가합니다.
    - 두 번째 파일부터는 헤더(첫 줄)를 제외한 데이터 줄만 `merged_data.csv`에 순서대로 추가합니다.
5.  **중복 제거**: 모든 데이터 취합 후, **행의 모든 열 값이 완전히 동일한** 중복 행을 제거합니다.
6.  **결과**: 중복이 제거된 최종 데이터가 `merged_data.csv`에 저장됩니다.

### 3.4. 완료 피드백

- 모든 과정이 성공적으로 완료되면, 사용자에게 "파일 병합이 완료되었습니다. `merged_data.csv` 파일이 생성되었습니다." 와 같은 확인 메시지를 표시합니다.

## 4. 기대 효과 (Expected Outcome)

- 분산된 데이터 파일을 수동으로 관리하는 불편함이 해소됩니다.
- 정제된 단일 데이터 파일을 통해 데이터 분석 및 처리가 용이해집니다.
