## ADDED Requirements
### Requirement: 자동 버전 확인
시스템은 시작 시 Github 저장소의 최신 릴리즈 버전을 확인해야 합니다.

#### Scenario: 새 버전 사용 가능
- **GIVEN** 현재 애플리케이션 버전이 `1.0.0`이고
- **WHEN** Github 최신 릴리즈 버전이 `1.1.0`일 때
- **THEN** 시스템은 새로운 업데이트가 있음을 감지하고 사용자에게 알려야 합니다.

#### Scenario: 이미 최신 버전 사용 중
- **GIVEN** 현재 애플리케이션 버전이 `1.1.0`이고
- **WHEN** Github 최신 릴리즈 버전이 `1.1.0`일 때
- **THEN** 시스템은 조용히 시작 프로세스를 계속 진행해야 합니다.

### Requirement: 업데이트 설치
사용자가 동의할 경우, 시스템은 새 버전을 다운로드하고 현재 버전을 대체한 후 자동으로 재시작해야 합니다.

#### Scenario: 사용자가 업데이트에 동의
- **GIVEN** 새로운 업데이트가 감지되었고
- **WHEN** 사용자가 업데이트를 진행하는 것에 동의할 때
- **THEN** 시스템은 새 버전의 파일을 다운로드해야 합니다.
- **AND** 시스템은 현재 애플리케이션을 종료하고 새 파일로 덮어써야 합니다.
- **AND** 업데이트가 완료된 후 시스템은 새로운 버전의 애플리케이션을 자동으로 재시작해야 합니다.
