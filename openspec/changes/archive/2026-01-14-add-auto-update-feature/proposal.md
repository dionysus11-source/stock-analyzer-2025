# 변경: 자동 업데이트 기능 추가

## Why
현재 애플리케이션은 사용자가 직접 `main.exe`를 다운로드하여 수동으로 업데이트해야 합니다. 자동 업데이트 기능을 추가하여 사용자가 항상 최신 버전을 사용하도록 보장하고 편의성을 개선합니다.

## What Changes
- 애플리케이션 시작 시 Github Release에 게시된 최신 버전을 확인하는 기능이 추가됩니다.
- 새로운 버전이 있을 경우, 사용자에게 알리고 업데이트를 진행할 수 있는 옵션을 제공합니다.
- 사용자가 동의하면 새 버전의 파일을 다운로드하고 기존 파일을 교체한 후 애플리케이션을 다시 시작합니다.

## Impact
- **Affected specs**: `auto-updater` (신규)
- **Affected code**:
    - `main.py`: 업데이트 확인 로직 호출
    - `auto_updater.py`: 새로운 모듈로 업데이트 로직 전체를 담당
    - `_version.py`: 애플리케이션 버전 관리
    - `make_exe_file.bat`: 빌드 프로세스에 버전 정보 포함 (필요시)
