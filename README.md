========================================
    주식 평가손익 분석기 v2.0
========================================

📁 파일 구성:
├── main.exe              # 실행파일 (빠른 실행)
├── main.py               # Python 소스코드
├── ui_components.py      # UI 컴포넌트
├── file_handler.py       # 파일 처리
├── data_analyzer.py      # 데이터 분석
├── requirements.txt      # Python 패키지 목록
├── TradingHistory.csv    # 샘플 데이터
├── run_analyzer.bat      # 메인 실행 스크립트 (권장)
├── run_exe.bat          # 실행파일 실행
├── run_python.bat       # Python 실행
└── README.txt           # 이 파일

🚀 실행 방법:

방법 1: 메인 스크립트 (권장)
  → run_analyzer.bat 더블클릭
  → 실행 방법 선택

방법 2: 직접 실행
  → main.exe 더블클릭 (가장 빠름)
  → 또는 run_exe.bat 실행

방법 3: Python으로 실행 (개발용)
  → run_python.bat 실행
  → 또는 명령프롬프트에서: python main.py

📋 기능:
- CSV 파일 드래그 앤 드롭
- 주식 거래 데이터 분석
- 평가손익 계산
- 통계 정보 표시
- 빠른 시작 속도 (0.1초)

⚡ 성능:
- 시작 속도: 0.1초 (최적화됨)
- 메모리 사용량: 낮음
- 파일 크기: 작음

🔧 문제 해결:
- Python 오류: Python 3.8+ 설치 필요
- 라이브러리 오류: run_python.bat이 자동 설치
- 실행파일 오류: main.exe 파일 확인
